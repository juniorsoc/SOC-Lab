# Tools Used in This Lab

| Tool | Purpose |
|---|---|
| Kali Linux | Attacker machine — Nmap, Hydra, Metasploit for generating test traffic |
| Ubuntu | Stable host for Splunk (SIEM) and Suricata (IDS) |
| Splunk | Log aggregation, correlation, and alert investigation |
| Suricata | Network intrusion detection (IDS) |
| Wireshark | Packet capture and traffic analysis |
| Nmap | Network scanning and reconnaissance (used offensively to generate detectable traffic) |
| Wazuh (planned) | Host-based intrusion detection / extended SIEM capabilities |
