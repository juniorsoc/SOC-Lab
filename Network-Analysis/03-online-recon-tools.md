# Online Tools for Recon and Analysis

| Tool | Purpose |
|------|---------|
| Shodan | Search engine for internet-connected devices — finds exposed ports/services |
| VirusTotal | Checks a file/URL/hash against many AV engines at once |
| NVD (National Vulnerability Database) | Authoritative source for CVE records + CVSS scores |
| Exploit-DB | Repository of proof-of-concept exploit code |
| GitHub | Security scanners, tools, CVE write-ups — often faster than official channels |

**Caution with PoC code:** not every published exploit is complete/reliable, some contain intentional flaws, and rarely a repo itself can be malicious. Review before running, avoid running unfamiliar scripts in production.
