# Intrusion Detection Systems (IDS)

Monitors a network/host for unauthorized activity — compares behavior against known malicious patterns or deviations from normal baseline. Examples: unauthorized device joining the network, a user touching a file they shouldn't have access to.

**Detection, not prevention.** Unlike a firewall, an IDS doesn't block anything — it flags and alerts. An IDS alert means something was observed, not that it was stopped.
